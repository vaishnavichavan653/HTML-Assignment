HTML text formatting tags are used to change the appearance and meaning of text. This page explains and demonstrates all commonly used text formatting tags.

Bold Text The <b> tag is used to display text in bold style.

Strong Text The <strong> tag is used to show important text.

Italic Text The <i> tag is used to display text in italic style.

Emphasized Text The <em> tag is used to emphasize text.

Underlined TextThe <u> tag is used to underline the text.

Highlighted Text The <mark> tag is used to highlight text.

Small Text The <small> tag is used to display smaller text.

Deleted Text The <del> tag shows deleted or removed text.

Inserted Text The <ins> tag shows newly added text.
